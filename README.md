# AI CV Scanner & TalentIQ

A smart, AI-powered recruitment system built with Next.js and n8n workflows. This project automates the CV screening process, compares candidates against Job Descriptions (JD), and features an AI chatbot for HR professionals to query candidate data in real-time.

## Features

### 1. Automated CV Scoring
- Upload multiple CVs (PDF, DOCX).
- Input a Job Description (JD) to set the context.
- The AI automatically extracts skills, experience, and calculates a match score for each candidate.

### 2. Candidate Dashboard
- **Table View**: Detailed list of all candidates and their metrics.
- **Card View**: Interactive cards with quick actions.
- **Actions**: View detailed profile, Shortlist, or Reject candidates.

### 3. AI Chatbot Assistant (RAG)
- Built-in AI chat interface connected to a vector database (Qdrant).
- Query candidate data using natural language (e.g., *"Find candidates with over 3 years of React experience"*).
- Summarize top candidate profiles instantly.

## Getting Started

### 1. Prerequisites
- Node.js (v18 or higher)
- An active [n8n](https://n8n.io/) instance (self-hosted or cloud)
- [Qdrant](https://qdrant.tech/) vector database (for chatbot RAG features)
- OpenAI API Key (or any LLM supported by n8n)

### 2. n8n Workflows Setup
The AI logic runs entirely on n8n. You need to import the workflows provided in the `workflow/` folder into your n8n instance:
1. **CV Processing Workflow**: Import `workflow/CV_Processing_LLM_Pipeline_(Final).json`. Set up the webhook URL and activate it.
2. **Chatbot Workflow**: Import `workflow/Chatbot_Web_Fixed.json`. Ensure it connects to your Qdrant vector database.
3. **Email Automation**: Import `workflow/Automated_Email_Candidates.json`.

*Make sure to note down the webhook URLs generated by n8n, as you will need them for the environment variables.*

### 3. Environment Variables
Create a `.env.local` file in the root directory based on your n8n configuration:

```env
# n8n Base Webhook URL (Replace with your instance URL)
N8N_WEBHOOK_BASE=https://your-n8n-instance.com/webhook

# n8n Authentication (If using header auth)
N8N_API_KEY=your_secure_api_key

# Specific Webhook Paths for features
N8N_SCORE_WEBHOOK_URL=https://your-n8n-instance.com/webhook/hr-cv
N8N_CHAT_WEBHOOK_URL=https://your-n8n-instance.com/webhook/chatbot

# Development Mode
NEXT_PUBLIC_USE_MOCK_DATA=false
```

### 4. Run the Application
```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Open the app
# Navigate to http://localhost:3000
```
