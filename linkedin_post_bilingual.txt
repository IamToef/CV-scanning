🚀 How I Automated CV Screening using an AI Agent & RAG (Next.js + n8n + Qdrant + Gemini)

Recruiters spend hours manually screening CVs. To solve this, I built recruitPRO — an open-source, AI-driven recruitment system that automates resume scanning, scoring, and semantic search.

How it works:
1️⃣ Next.js (Vercel): Modern UI for CV/JD uploads, candidate scoring, and RAG chat.
2️⃣ n8n Workflow (Docker): The "brain" coordinating file parsing and agentic flows without hardcoded backends.
3️⃣ Gemini API (Google AI Studio): Extracts structured JSON (evaluating strengths, gaps, and match scores) 100% free.
4️⃣ Qdrant (Vector DB): Stores candidate vectors for semantic RAG search (e.g., "Find a React developer with 2+ years of experience").

Key Lessons:
💡 Visual debugging in n8n makes prompt validation and JSON schema alignment with Next.js super fast.
💡 Vector DB semantic matching is far superior to keyword search for candidate sourcing.

🚀 Tự Động Hóa Quy Trình Lọc CV Bằng AI Agent & Kiến Trúc RAG (Next.js + n8n + Qdrant + Gemini)

Sàng lọc CV và tìm kiếm ứng viên thủ công luôn tiêu tốn hàng giờ của HR. Mình đã phát triển recruitPRO — hệ thống tuyển dụng ứng dụng AI tự động hóa hoàn toàn quy trình xử lý, chấm điểm CV và tìm kiếm ngữ nghĩa.

Cách thức hoạt động:
1️⃣ Next.js (Vercel): Giao diện upload CV/JD, xem kết quả chấm điểm và chat RAG.
2️⃣ n8n Workflow (Docker): "Bộ não" tiếp nhận webhook, xử lý văn bản PDF và điều phối tác vụ AI trực quan.
3️⃣ Gemini API (Google AI Studio): Trích xuất dữ liệu JSON có cấu trúc (phân tích thế mạnh, điểm yếu, chấm điểm CV/JD) hoàn toàn miễn phí.
4️⃣ Qdrant (Vector DB): Mã hóa và lưu trữ vector hồ sơ ứng viên phục vụ tìm kiếm thông minh bằng ngôn ngữ tự nhiên.

Bài học rút ra:
💡 Nền tảng Low-code (n8n) giúp rút ngắn 80% thời gian phát triển backend và tối ưu hóa luồng gọi LLM.
💡 Tìm kiếm ngữ nghĩa (Qdrant) giúp HR tìm kiếm ứng viên thông minh hơn tìm kiếm từ khóa truyền thống.
